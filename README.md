## HTML

Lenguaje que leen los browsers para pintar páginas de colores

### Tags

  - div
    - ocupa todo el ancho `width` posible
    - empieza en un renglón nuevo
      - `display: block`

  - h1, h2, h3, h4, h5
    - Tamaños de titulos

  - p (paragraph)
    - Margenes `margin` arriba y abajo
